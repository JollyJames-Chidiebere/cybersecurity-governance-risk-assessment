## Overview

This project demonstrates a comprehensive cybersecurity risk assessment conducted for **Toolbox International Limited**, a fictional multinational artisan marketplace operating across Africa.

The assessment evaluates the organization's information assets, identifies cybersecurity threats and vulnerabilities, prioritizes business risks, assesses governance maturity using **COBIT 2019**, recommends security improvements based on the **CIS Critical Security Controls v8**, and aligns privacy practices with **PIPEDA**.

The objective of this project was to demonstrate a structured approach to governance, risk management, and cybersecurity compliance while developing practical recommendations to improve the organization's overall security posture.

---

## Project Objectives

- Perform an enterprise cybersecurity risk assessment
- Identify and prioritize critical business assets
- Analyze cybersecurity threats and vulnerabilities
- Evaluate organizational risk using a qualitative risk matrix
- Assess governance maturity using COBIT capability levels
- Recommend security controls using CIS Controls v8
- Evaluate privacy compliance using PIPEDA principles
- Develop governance and security improvement recommendations

---

## Business Scenario

Toolbox International Limited is a fictional multinational marketplace connecting customers with skilled artisans across several African countries.

The organization relies heavily on its web platform, mobile applications, customer databases, and internal IT infrastructure to deliver services. Due to the sensitivity of customer and organizational information, a structured security assessment was performed to identify business risks and recommend governance improvements.

## Key Risks Identified

| Risk | Priority |
|-------|----------|
| Unauthorized Access | Critical |
| Insider Threat | Critical |
| Poor Incident Response | Critical |
| System Failure | High |
| Poor Backup Strategy | High |
| Human Error | High |
| Phishing Emails | Medium |
| Natural Disaster (Flood) | Medium |

---

## Key Recommendations

- Strengthen Identity and Access Management
- Implement Multi-Factor Authentication (MFA)
- Improve Backup and Recovery Procedures
- Develop a Formal Incident Response Plan
- Establish Continuous Security Awareness Training
- Maintain Accurate Hardware Asset Inventory
- Implement DNS Filtering and Email Security Controls
- Improve Governance Maturity through COBIT Processes
